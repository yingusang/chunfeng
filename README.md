# 🌸 春风 - 六爻占卜应用

**遇事不决，可问春风**

一个基于传统六爻占卜的决策辅助工具，提供卦象解析和决策建议。

## ✨ 特点

- 📿 完整的六爻占卜逻辑
- 🔮 64卦象详细解析
- 💡 明确的决策建议
- 📜 历史记录保存
- 🎨 优雅的界面设计
- 📱 响应式布局，支持手机访问

## 🚀 在线访问

访问地址：[https://yingusang.github.io/chunfeng/]

## 💻 本地使用

直接在浏览器中打开 `chunfeng.html` 文件即可使用。

## 📖 使用方法

1. 输入你的问题（可选）
2. 点击"开始占卜"按钮
3. 查看卦象和解析结果
4. 可以查看历史占卜记录

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript
- 无需后端服务器
- 使用 localStorage 保存历史记录
